安裝套件​

  • 光碟片不能移除
  • 先到 yast 的 software repositories 所有套件都先 enable
# 新增 nano 套件
rancher@rke2-1:~> wget https://download.opensuse.org/repositories/editors/SLE_15/x86_64/nano-7.2-178.1.x86_64.rpm
rancher@rke2-1:~> sudo rpm -i nano-7.2-178.1.x86_64.rpm
# 安裝 podman
rancher@lb:~> sudo zypper ar --refresh https://download.opensuse.org/repositories/Virtualization:/containers/SLE_12_SP5/Virtualization:containers.repo

rancher@lb:~> sudo zypper ref

rancher@lb:~> sudo zypper in podman
# 安裝 brctl & ifconfig 網路工具
$ sudo zypper addrepo https://download.opensuse.org/repositories/network:utilities/SLE_15_SP4/network:utilities.repo

$ sudo zypper refresh

$ sudo zypper in -y net-tools-deprecated bridge-utils

rancher@rke2-1:~> sudo zypper install nano net-tools-deprecated tree
  • 安裝 locate
$ sudo zypper in -y mlocate

# 需要先更新資料庫
$ sudo updatedb

$ locate passwd

註冊​

# 檢查自己是否有註冊
$ sudo SUSEConnect -s
[{"identifier":"SLES","version":"15.4","arch":"x86_64","status":"Not Registered"},{"identifier":"sle-module-basesystem","version":"15.4","arch":"x86_64","status":"Not Registered"},{"identifier":"sle-module-server-applications","version":"15.4","arch":"x86_64","status":"Not Registered"},{"identifier":"sle-module-containers","version":"15.4","arch":"x86_64","status":"Not Registered"}]
# 註冊，前面是註冊code，後面是公司信箱
$ sudo SUSEConnect -r REGISTRATION_CODE -e EMAIL_ADDRESS

$ sudo SUSEConnect -r INTERNAL-USE-ONLY-8a84-3057 -e andy.wu@suse.com

# 清掉註冊
$ sudo SUSEConnect -d

# 這是內部的 rmt server
# rmt 也是 DNS sever ip:10.12.0.20 or 192.168.11.20 要先加才能解析得到，目前只有 sles 15 sp4 的 patch
$ sudo SUSEConnect --url=http://rmt.suselab.tw

repo enable​

$ sudo zypper modifyrepo -e '儲存庫名稱'

把 rancher 加入 wheel 群組​

rancher@rke2-1:~> sudo usermod -a -G wheel rancher
# 將 andy 從 wheel 群組移除
$ sudo gpasswd -d andy wheel

手動設定網路 config 檔​

$ sudo vim /etc/sysconfig/network/ifcfg-eth0
IPADDR='192.168.11.109/24'
BOOTPROTO='static'
STARTMODE='auto'

$ sudo vim /etc/sysconfig/network/ifroute-eth0
default 192.168.11.254 - eth0
